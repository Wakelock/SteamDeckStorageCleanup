# Steam Deck Storage Cleanup
Delete unnecessary files from your Steam Deck

## Usage
Minimal cleanup: `/home/deck/Documents/SteamDeckStorageCleanup.sh`

Maximal cleanup `/home/deck/Documents/SteamDeckStorageCleanup.sh -sudo`

Check for updates: `/home/deck/Documents/SteamDeckStorageCleanup.sh -update`

Note: using maximal cleanup mode makes your system less foolproof by making it read-write. If you're insecure, you can re-enble read-only with `sudo steamos-readonly enable`.
